# Images

Drop your photos here. The site will automatically pick them up by filename.

| File            | Used in                     | Recommended shot                         |
| --------------- | --------------------------- | ---------------------------------------- |
| `hero.jpg`      | Full-screen hero background | Aerial houseboat / Shuswap Lake panorama |
| `houseboat.jpg` | Intro photo grid (large)    | Side view of the houseboat               |
| `lake1.jpg`     | Intro photo grid (left)     | Water activities / group shot            |
| `lake2.jpg`     | Intro photo grid (right)    | Lake / landscape                         |
| `activity.jpg`  | Everyday section (circular) | Action shot — tubing, wakeboarding, etc. |
| `closing.jpg`   | Closing hero background     | Sunset on the water                      |

All images should be JPG or WebP. Landscape orientation works best.
Aim for 1920px wide for full-bleed backgrounds, 800–1200px for smaller slots.
